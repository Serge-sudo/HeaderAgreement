## Функциональность

* Конструктор `Q()`

* Конструктор копирования  `Q(const Q&)`

    - Базовое требование курса

    - Проверка на независимость копий


* Оператор присваивания   `Q& operator =`

    - Базовое требование курса

    - `q=q`

* Move конструктор

    - Базовое требование курса

* Move =

    - Базовое требование курса 

* `pop()`

    - Из пустой не кидает ничего

* `front()`

    - Кидает если вызвать у пустой
